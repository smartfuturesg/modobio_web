odyssey.constants module¶

odyssey.constants.COUNTRIES¶

Names of countries and their 2-letter codes.

A tuple of tuples: ((‘US’, ‘United States of America’), …)

odyssey.constants.USSTATES¶

Names of US states and their 2-letter codes.

A tuple of tuples: ((‘AZ’, ‘Arizona’), …)

odyssey.constants.GENDERS¶

Genders and a 1-letter abbreviation.

A tuple of tuples: ((‘f’, ‘female’), …)

odyssey.constants.CONTACT_METHODS¶

Available methods to contact a client.

A tuple of tuples: ((0, ‘phone’), …)

odyssey.constants.YESNO¶

Boolean options of ‘Yes’ or ‘No’.

This can be used to link a wtforms.fields.RadioField form field to a sqlalchemy.types.Boolean database column. Use in conjunction with BOOLIFY:

name = RadioField(‘label’, choices=YESNO, coerce=BOOLIFY)

odyssey.constants.BOOLIFY¶

Convert a number into a boolean.

A POST request always returns a string. A sqlalchemy.types.Boolean database column expects a Python True or False value. Use BOOLIFY() to convert the returned string value ‘0’ or ‘1’ into False or True, respectively. Use in conjuction with YESNO:

name = RadioField(‘label’, choices=YESNO, coerce=BOOLIFY)
