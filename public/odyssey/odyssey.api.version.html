odyssey.api.version¶

class odyssey.api.version.VersionEndpoint(api=None, *args, **kwargs)¶

Bases: flask_restx.resource.Resource

get()¶

Returns API version in response to a GET request.

Version number can come from odyssey.version, generated by setuptools_scm, or from environmental variable API_VERSION. The environmental variable takes precedence.

Returns

str – API version number
