odyssey.utils.constants.YESNO = ((0, 'No'), (1, 'Yes'))¶

Boolean options of ‘Yes’ or ‘No’.

This can be used to link a wtforms.fields.RadioField form field to a sqlalchemy.types.Boolean database column. Use in conjunction with BOOLIFY.:

name = RadioField('label', choices=YESNO, coerce=BOOLIFY)
Type

tuple(tuple)

odyssey.utils.constants.BOOLIFY(x)¶

Convert a number into a boolean.

A POST request always returns a string. A sqlalchemy.types.Boolean database column expects a Python True or False value. Use BOOLIFY() to convert the returned string value ‘0’ or ‘1’ into False or True, respectively. Use in conjunction with YESNO.:

name = RadioField('label', choices=YESNO, coerce=BOOLIFY)
Type

callable()

odyssey.utils.constants.USER_TYPES = ('staff', 'staff_self', 'client')¶

The general category of odyssey.models.user.User consists of subtypes. This constant lists the types resource access contexts APIs will fit under.

‘client’ - client may acess resource only for themselves. ‘staff’ - may access resource for clients who have given the staff member explicit permission ‘staff_self’ - staff may access only thier own resource (profile, role assignment etc.)

Type

tuple
