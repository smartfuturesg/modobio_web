available_timeslots(target_time_range: Tuple[flask_restx.fields.DateTime, flask_restx.fields.DateTime], location_id: int, clinician_id: str = '')

Query wheel’s API to find timeslots for a specific datetime range. Wheel URI: /v1/consult_rates/<consult_rate>/timeslots

Parameters:

  • target_time_range (tuple(datetime.datetime, datetime.datetime)) – tuple containing the start and end datetimes of the target booking window in UTC

  • location_id (int) – where the client is located. Converted to state abbreviation for wheel request

  • clinician_id (str) – optional wheel_clinician_id of the clinician

Returns:

dict – available time increments: {user_id: [booking_availability_ids]}

openings(target_time_range, location_id, clinician_id='')

Query wheel’s API to find timeslots for a specific datetime range. Wheel URI: /v1/consult_rates/<consult_rate>/openings
