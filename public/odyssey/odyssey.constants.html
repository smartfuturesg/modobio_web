odyssey.constants.YESNO¶

Boolean options of ‘Yes’ or ‘No’.

This can be used to link a wtforms.fields.RadioField form field to a sqlalchemy.types.Boolean database column. Use in conjunction with BOOLIFY.:

name = RadioField('label', choices=YESNO, coerce=BOOLIFY)
Type

tuple(tuple)

odyssey.constants.BOOLIFY(x)¶

Convert a number into a boolean.

A POST request always returns a string. A sqlalchemy.types.Boolean database column expects a Python True or False value. Use BOOLIFY() to convert the returned string value ‘0’ or ‘1’ into False or True, respectively. Use in conjunction with YESNO.:

name = RadioField('label', choices=YESNO, coerce=BOOLIFY)
