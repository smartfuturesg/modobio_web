class odyssey.api.system.routes.SystemDeleteUserApi(*args, **kwargs)

Bases: odyssey.utils.base.resources.BaseResource

Endpoint for the system admin to delete a user

delete(user_id)

This endpoint can be used to instantly delete a user by the system admin. Note that unlike the self-deletion process that a user can initiate, this does not have the step of the user being makred as ‘closed’ for 30 days before being deleted by the automated system.

The system admin should specify which portion of the user’s account should be deleted by supplying either ‘client’, ‘staff’, or ‘both’ in the delete_type arg
