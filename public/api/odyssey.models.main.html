odyssey.models.main module¶

General database tables for the operation of the Modo Bio Staff application. There is no specific table name prefix for the tables in this module.

class odyssey.models.main.Staff(**kwargs)¶

Bases: sqlalchemy.ext.declarative.api.Model

Staff member information table.

This table stores basic login information regarding Modo Bio staff members.

email¶

Email address of staff member, used as login.

Type

str, max length 50, non-null, unique

firstname¶

First name of staff member.

Type

str, max length 50, non-null

fullname¶

Full name of staff member. Combination of firstname + lastname.

Type

str, max length 100

is_admin¶

Indicates whether staff member is an administrator, i.e. has the ability to add/delete members to the Staff table.

Type

bool, default = False

lastname¶

Last name of staff member.

Type

str, max length 50, non-null

password¶

Hashed password string, as generated by werkzeug.security.generate_password_hash().

Type

str, max length 128

staffid¶

Staff member ID number

Type

int, primary key, autoincrement
