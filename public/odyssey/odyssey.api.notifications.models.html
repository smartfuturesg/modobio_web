odyssey.api.notifications.models module¶

Database tables for the notifications section of the Modo Bio API. All tables in this module are prefixed with ‘Notifications’.

time_to_live¶

The lifetime of the notification in seconds. If None, the notification does not expire. If both time_to_live and expires are set, then time_to_live is ignored.

Type

int or None

Unit

seconds

expires¶

The time this notification will expire. If time_to_live is set and expires is not set, then expires will be calculated from time_to_live. If both are None, the notification will not expire.

Type

datetime or None

read¶

Whether the user has read this notification.

Type

bool

deleted¶

Whether the user has deleted this notification. This will not remove the entry from the database, but can be used to hide this notification in a UI.

Type

bool
