Staff member information table.

This table stores basic login information regarding Modo Bio staff members.

token¶

API authentication token

Type

str, max length 32, indexed, unique

token_expiration¶

token expiration date

Type

datetime

staffid¶

Staff member ID number

Type

int, primary key, autoincrement

is_admin¶

Indicates whether staff member is an administrator, i.e. has the ability to add/delete members to the Staff table.

Type

bool, default = False

is_system_admin¶

Indicates whether staff member is a system administrator. There should be very few of these roles given out.

Type

bool, default = False

access_role¶

Indicates the content access role of the staff member. roles include: client services (cs), pt, doctor, data

Type

str, default = cs
