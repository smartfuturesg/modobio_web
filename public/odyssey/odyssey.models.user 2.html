static generate_modobio_id(firstname: str, lastname: str, user_id: int) → str¶

Generate the user’s mdobio_id.

The modo bio identifier is used as a public user id, it can also be exported to other healthcare providers (clients only). It is made up of the firstname and lastname initials and 10 random alphanumeric characters.

Parameters

  • firstname (str) – Client first name.

  • lastname (str) – Client last name.

  • user_id (int) – User ID number.

Returns

str – Medical record ID
