class odyssey.models.intake.ClientInfo(**kwargs)¶

Bases: sqlalchemy.ext.declarative.api.Model

Client information table

This table stores general information of a client. The information is taken only once, during the initial consult. The primary index of this table is the clientid number. Many other tables in this database refer to the clientid number, so a new client MUST be added to this table first, in order to generate the clientid number.

Client full address.

Generate by combining street, city, state, zipcode, and country.
