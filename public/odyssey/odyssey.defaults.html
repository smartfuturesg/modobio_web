odyssey.defaults¶

Configuration defaults

This file defines the configuration defaults for the entire API. It is loaded by odyssey.config

odyssey.defaults.FLASK_DEV = 'local'¶

Determines the development environment. Can be ‘local’ or ‘remote’.

odyssey.defaults.S3_BUCKET_NAME = '/var/folders/sb/369pt0b13hg18d1cdbgbgp5h0000gq/T/tmpfyuzy9v2'¶

Name of the AWS S3 bucket where files are stored. For FLASK_DEV=local it is set to a temporary directory.

odyssey.defaults.WHOOSHEE_DIR = None¶

Directory where whooshee stores persistent index. Defaults to local directory if unset.

odyssey.defaults.DB_FLAV = 'postgresql'¶

Type of database.

odyssey.defaults.DB_USER = ''¶

Username for database.

odyssey.defaults.DB_PASS = ''¶

Password for database.

odyssey.defaults.DB_HOST = 'localhost'¶

Hostname for database.

odyssey.defaults.DB_NAME = 'modobio'¶

Name of database.

odyssey.defaults.OURA_CLIENT_ID = ''¶

Client ID for Oura Ring API.

odyssey.defaults.OURA_CLIENT_SECRET = ''¶

Client secret for Oura Ring API.

odyssey.defaults.OURA_AUTH_URL = 'https://cloud.ouraring.com/oauth/authorize'¶

Authorization URL for Oura Ring API.

odyssey.defaults.OURA_TOKEN_URL = 'https://api.ouraring.com/oauth/token'¶

Token URL for Oura Ring API.

odyssey.defaults.SECRET_KEY = 'dev'¶

Flask secret key, used to encrypt sessions amongst other things. See https://stackoverflow.com/questions/22463939/demystify-flask-app-secret-key

odyssey.defaults.SQLALCHEMY_TRACK_MODIFICATIONS = False¶

Turn this off to prevent warnings.

odyssey.defaults.TESTING = False¶

Special mode for running tests.

odyssey.defaults.WTF_CSRF_ENABLED = True¶

Cross Site Request Forgery (CSRF) on forms, disable when testing. Not needed on API, to be removed with Flask app removal.
