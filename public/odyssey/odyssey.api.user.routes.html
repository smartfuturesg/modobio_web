class odyssey.api.user.routes.PasswordResetEmail(api=None, *args, **kwargs)¶

Bases: flask_restx.resource.Resource

Password reset endpoints.

post()¶

begin a password reset session. Staff member unable to log in will request a password reset with only their email. Emails will be checked for exact match in the database to a staff member.

If the email exists, a signed JWT is created; encoding the token’s expiration time and the user_id. The code will be placed into this URL <url for password reset> and sent to a valid email address. If the email does not exist, no email is sent. response 200 OK
