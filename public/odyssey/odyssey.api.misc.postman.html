odyssey.api.misc.postman¶

Download API as a Postman collection.

This endpoint is not part of the flask-restx based API, instead it is implemented as a plain Flask route. It is only available in developer mode, when run with FLASK_DEV=local.

When accessing this endpoint at http://localhost/api/postman, it returns a JSON file for download. That JSON file can be imported into Postman and provides a collection containing all endpoints in this API.
